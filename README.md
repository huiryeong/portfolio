## 🛠 데이터·AI 활용 능력

### Transcreate — 리스크 사전 플래깅 도구
메시지가 타깃 환경에 맞지 않을 때 생길 수 있는 위험 요소를 사전에 자동 플래깅하는 AI 도구.
문제 정의부터 개발·배포까지 직접 수행했습니다. **사고가 발생하기 전에 리스크를 잡아내는 설계**는 협력사 운영의 사전 리스크 관리에 그대로 적용됩니다.

**→ [라이브 데모](https://transcreate-66856309088.asia-east1.run.app/)**

- Google AI Studio (Gemini API) 기반 · Cloud Run 배포
- 위험 요소 자동 플래깅

### Concept Builder — 정량 평가 기반 의사결정 도구
하나의 답이 아니라 서로 대조적인 3가지 대안을 동시에 생성하고, 정량 점수(Fit Score)로 비교·평가.
**여러 선택지를 지표로 비교해 우선순위를 정하는 사고**는 협력사 평가·정책 우선순위 결정에 적용됩니다.

**→ [라이브 데모](https://service-885528335121.asia-east1.run.app/)**

- Territory별 대안 + 정량 Fit Score 생성

## 💻 데이터 기반 프로젝트

"왜 그렇게 판단했는지"를 함께 제시하는 설명 가능한(XAI) 데이터 판정 시스템.
수치에 근거를 더해 **상대가 납득하는 판단**을 구현 — 22개 팀 중 2위 수상, CMC 논문 제출.
→ 협력사 평가·피드백을 납득 가능하게 전달하는 일과 직결됩니다.

### 싱크홀 감지 IoT 시스템
도심 싱크홀을 조기 감지하는 라즈베리파이 기반 IoT 시스템.
**일상을 떠받치는 인프라를 데이터로 안정적으로 유지하는 관점** — 변수가 큰 배달 인프라 운영과 닿아 있습니다.
