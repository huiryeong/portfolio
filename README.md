글로벌 광고 카피를 단순 번역이 아닌, 타깃 시장의 문화 코드에 맞게 재창작하는 AI 툴.
직역 시 발생할 수 있는 문화적 위험 요소를 사전에 플래깅합니다.

**→ [라이브 데모](https://transcreate-66856309088.asia-east1.run.app/)**

- Google AI Studio (Gemini API) 기반
- 문화적 리스크 플래깅 기능
- Nike, Apple, Airbnb 등 실제 글로벌 브랜드 케이스 검증

### Campaign Concept Builder — AI 캠페인 기획 툴
브랜드 브리프를 입력하면 대행사 수석 CD처럼 상호 대조적인 3가지 크리에이티브 루트를 자동 생성.
